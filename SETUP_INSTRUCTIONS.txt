ИНСТРУКЦИИ ПО НАСТРОЙКЕ ИГРЫ - АВТОЗАПУСК СЦЕНЫ "GAME"
═══════════════════════════════════════════════════════════════════

ПРОБЛЕМЫ ИСПРАВЛЕНЫ:
---------------------
1. ✓ Создан GameStartup.cs - автоматически переключает на сцену game при нажатии Play
2. ✓ Отключен Camera_1.cs - конфликтовал с PlayerController в Run.cs
3. ✓ Проверены остальные скрипты на ошибки

ЧТО НУЖНО СДЕЛАТЬ В UNITY:
---------------------------

1. ОТКРОЙТЕ UNITY EDITOR
   - Подождите, пока Unity скомпилирует все скрипты
   - Проверьте Console (Window → General → Console)
   - Убедитесь, что нет красных ошибок

2. ПРОВЕРЬТЕ BUILD SETTINGS
   - File → Build Settings
   - Убедитесь что сцена "game.unity" стоит ПЕРВОЙ в списке
   - Если нет - перетащите её на первое место

3. ПРОВЕРЬТЕ ИГРОВОЙ ОБЪЕКТ "PLAYER"
   - Найдите игрока в иерархии сцены game
   - У него должен быть компонент "Player Controller (Script)" (из Run.cs)
   - Если есть компонент "Camera 1 (Script)" - УДАЛИТЕ его!

4. ПРОВЕРЬТЕ КАМЕРУ
   - Камера должна быть ДОЧЕРНИМ объектом игрока
   - У камеры НЕ должно быть компонента "Camera 1 (Script)"

5. ТЕСТИРОВАНИЕ
   - Откройте ЛЮБУЮ сцену (например lvl2.unity или lvl3.unity)
   - Нажмите Play (►)
   - Игра должна автоматически переключиться на сцену game!
   - В Console должно появиться сообщение:
     "[GameStartup] Auto-switched to game scene for Play Mode"

КАК РАБОТАЕТ СИСТЕМА:
----------------------
• GameStartup.cs использует [InitializeOnLoad] для отслеживания входа в Play Mode
• При нажатии Play проверяется текущая сцена
• Если это не "game" - автоматически открывается game.unity
• Работает ТОЛЬКО в Unity Editor, не влияет на готовую сборку игры

ЕСЛИ ВСЁ РАВНО НЕ РАБОТАЕТ:
----------------------------
1. Проверьте Console на ошибки компиляции
2. Убедитесь что файл называется именно "game.unity" (маленькими буквами)
3. Попробуйте закрыть и заново открыть Unity
4. Проверьте что игрок имеет тег "Player" (Inspector → Tag)

РЕЗЕРВНЫЕ КОПИИ:
----------------
• Camera_1.cs.bak - оригинальная версия Camera_1.cs (если нужно восстановить)

КОНТАКТЫ И ПОДДЕРЖКА:
----------------------
Если есть вопросы - проверьте Console в Unity на наличие ошибок и предупреждений.
